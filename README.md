## Arquitectura de la Startup
Este proyecto implementa una solución escalable utilizando Programación Orientada a Objetos y Bases de Datos Relacionales.

### Componentes:
- **POO:** Uso de herencia y encapsulamiento para Clientes y Productos.
- **Persistencia:** Base de datos SQLite con integridad referencial.
- **Robustez:** Control de excepciones en todas las operaciones CRUD.
